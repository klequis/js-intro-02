JavaScript Stop Light

Create a working stop light by manipulating the red, green and yellow images.

Overview

The /images folder contains one 'on' and one 'off' image for each of the 3 stop light colors. The HTML page already contains 3 images. Manipulate the source property of each image switch the state of the 3 light colors.

Step-by-Step Instructions

  1. Write a function named changeLight() that takes on parameter which is the color of the light which should be on.
  2. In the HTML add a click event to each button that calls the changeLight() function passing it the color of the light to turn on.

    3. Extra Credit

    Make the light operate off of a timer
