Using getElementById()

Goal

Use getElementById() to get a reference to elements on the page and demonstrate that the reference works by changing the element's style.

Rules

  1. Use only HTML, CSS & JavaScript. Do not use jQuery.

Resources

  • Function (MDN)
  • getElementById() (MDN)
  • HTMLElement.style (MDN)

Instructions

  1. Inside the div with class 'wrapper', create a div with id of 'div1' and put the text '#div1' inside it.
  2. Below #div1, create a div with id of 'div2' and put the text '#div2' inside it.
  3. Create a function named main() that takes one parameter, the name of a div. Have the body of main() get a reference to the div per the passed-in parameter and set its background color to 'red'.
  4. Call main() from within you script.js file. There is no need to create UI for this exercise other than the two divs.
  5. Calling main() while passing 'div1' should make #div1 red and calling main() while passing 'div2' should make #div2 red.
