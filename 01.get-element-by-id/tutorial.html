Instructions

  2. Inside of the div 'wrapper' create divs an id of 'div2' and put the text '#div2' inside it.
  3. Create a function named main() takes one parameter, the name of a div. Have the body of main() get a reference to the div per the passed-in parameter and set its background color to 'red'
  4. Calling main() function with 'div1' should make #div1 red and calling main() with 'div2' should make #div2 red.
