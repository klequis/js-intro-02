Using addEventListener()

Goal

Create a button and connect it to an event handler. Clicking the button will write 'You clicked me' to the page.

Rules

  • Use only HTML, CSS & JavaScript. Do not use jQuery.
  • The event handler must be connected in JavaScript. Do not use onClick="" in HTML

Resources

  • getElementById() (MDN)
  • addEventListener() (MDN)
  • appendChild() (MDN)
  • Bootstrap 3.3/CSS

Instructions

  1. Inside of the div with class 'wrapper', create the 'Click Me' button with the bootstrap button classes that will make it green.
  2. Write a function named main() that attaches an event listener to the button. Have the event listener call a function named buttonClicked().
  3. Write the buttonClicked() function to write "You clicked me" to the body of the page each time the button is clicked, without overwriting the result of previous clicks.
  4. Make sure your Main() function is called when your script file loads.
