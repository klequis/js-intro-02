JavaScript Stop Light - Coding Challenge

Goal

Create a stop light that is operated by buttons.

Rules

  • Use only HTML, CSS & JavaScript. Do not use jQuery.
  • The event handler must be connected in JavaScript. Do not use onClick="" in HTML

Resources

  • event.target (MDN)

Instructions

  1. This project has a /images folder which contains one 'on' and one 'off' image for each of the 3 stop light colors. Manipulate the source property of elements to switch the state of the 3 light colors.
  2. Write a function named changeLight() that takes one parameter which is the color of the light to be turned-on and changes which pictures are displayed sot that the the correct color light is lit and the others are off.

    3. Extra Credit

    Use a timer to operate the light
