Light On/Off

Make the lightbulb turn on and off by manipulating the image's source property.

Instructions - Brief

  • Add to the html page a button that says 'Toggle Light' and an image with lightbulb-off.svg
  • Create a main funciton that attaches an event to the Toggle Light button
  • Create a toggle light function that switches the 'on' image to the 'off' image

Instructions - Detail

  1. Inside of div 'wrapper' add a 'Toggle Light' button.
  2. Below the button, add /images/lightbulb-off.svg to the page.
  3. Create a function named main() that attaches an event listener to the button.
  4. Create a function named toggleLight() that toggles between the 'on' and 'off' images.

Extra Credit

If your toggleLight() function depends on a path that will not work on a machine other than the one you are currently working on, then write a function that parses that string into something that will work on all machines.
