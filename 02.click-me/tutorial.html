Click Me - Coding Challenge

Goal

Connect a button to an event and print 'You clicked me' to the screen.

Rules

  • This is a JavaScript lesson so do not use jQuery

Hints

  • getElementById() MDN
  • addEventListener() MDN
  • appendChild() MDN
  • Bootstrap 3.3/CSS

Step-by-Step Instructions

  1. Inside of the div with class 'wrapper', create the 'Click Me' button with the bootstrap button classes that will make it green.
  2. Write a function named main() that attaches an event listener to the button. The event listener should call a function named buttonClicked()
  3. Write the buttonClicked() function which writes "You clicked me" to the body of the page each time the button is clicked without overwriting the result of previous clicks.
  4. Make sure your Main() function is called when your script file loads.
