Light On/Off - Coding Challenge

Goal

Make the lightbulb turn on and off by manipulating the image's source property.

Rules

  • Use only HTML, CSS & JavaScript. Do not use jQuery.
  • The event handler must be connected in JavaScript. Do not use onClick="" in HTML

Resources

  • The best resource is 'MDN web docs' (Mizilla Developer Network web docs.). Use Google search. Enter a search string followed by 'MDN' and you will usually find what you need.

Instructions - Detail

  1. Inside of the div with class 'wrapper' add a button with text 'Toggle Light'.
  2. Below the button, add /images/lightbulb-off.svg to the page.
  3. Create a function named main() which attaches an event listener to the button.
  4. Create a event handler function named toggleLight() that toggles between the 'on' and 'off' images.

Extra Credit

If your toggleLight() function depends on a path that will not work on a machine other than the one you are currently working on, write a function that parses that string into something that will work on all machines.
